## 기술적 쟁점
- 통계와 같은 그래프는 템플릿으로 대체
  - 이미 만들어진 템플릿에 데이터를 적용할 수 있는가
- 기존에 학습한 방식을 응용하여 얼마나 빠르고 간결하게 구현할 수 있는가
- 디자인에 너무 신경을 쓰지 않기 위해 간결한 디자인으로 빠르게 구현
